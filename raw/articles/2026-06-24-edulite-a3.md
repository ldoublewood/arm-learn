## 项目概述

EL-A3 是一款 7 自由度桌面机械臂（6 个臂关节 + 1 个夹爪），由两个独立子项目组成：纯 Python SDK 和 ROS2 控制系统。设计理念是将 SDK 与 ROS 解耦，用户可按需选择使用方式。

## 子项目结构

| 子项目 | 路径 | 说明 |
|--------|------|------|
| **Python SDK** | `el_a3_sdk/` | 纯 Python SDK，Direct CAN 通信，多臂管理，Pinocchio 动力学 |
| **ROS2 控制系统** | `el_a3_ros/` | ros2_control 硬件接口，MoveIt2 运动规划，URDF 描述 |
| **硬件制造资料** | `hardware/` | STEP 模型、3MF 打印文件、线束图纸、PCB 资料、组装 SOP |

## 硬件规格

| 项目 | 规格 |
|------|------|
| **机械臂** | EL-A3 7-DOF（6 臂关节 + L7 夹爪） |
| **电机配置** | L1-L3: Robstride RS00 ×3（±14 Nm），L4-L7: EL05/RS05 ×4（±6/±5.5 Nm） |
| **CAN 适配器** | CANdle / gs_usb 兼容设备 |
| **电源** | 24V / 48V 直流电源 |
| **PC** | Ubuntu 22.04+ x86_64 |

## 关键技术特点

### Python SDK
- 直接 CAN 通信，无需 ROS 依赖
- Pinocchio 动力学库集成（正/逆运动学、重力补偿）
- 多机械臂管理支持
- 内置 PyQt6 Debugger GUI：3D URDF 可视化（PyVista/VTK）、关节拖拽控制、实时监控

### ROS2 控制系统
- ros2_control 硬件接口
- MoveIt2 运动规划
- pick_ik 逆运动学求解器
- 手柄（joystick）控制支持

### 硬件制造
- STEP 格式 3D 模型（Git LFS 管理）
- 3MF 打印文件
- 线束工程图纸
- PCB 设计资料
- 装配 SOP（标准作业程序）
